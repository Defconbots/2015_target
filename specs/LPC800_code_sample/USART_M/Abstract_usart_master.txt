Purpose:
Example program for the USART periperals (synchronous MASTER mode) to configure various modes

Running mode:
* Compile, Flash the program and reset.
* Default project target set to USART_Release (exectuing from flash)
* Can the run project from RAM by setting project target to USART_Debug

Note:
Tested on LPC800 LPCXpresso Board
LPC800 running at 24 MHz
USART0 is configured
